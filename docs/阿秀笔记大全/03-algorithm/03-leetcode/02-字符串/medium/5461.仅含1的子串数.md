## [5461. 仅含 1 的子串数](https://leetcode-cn.com/problems/number-of-substrings-with-only-1s/)



给你一个二进制字符串 `s`（仅由 '0' 和 '1' 组成的字符串）。

返回所有字符都为 1 的子字符串的数目。

由于答案可能很大，请你将它对 10^9 + 7 取模后返回。

 

**示例 1：**

```
输入：s = "0110111"
输出：9
解释：共有 9 个子字符串仅由 '1' 组成
"1" -> 5 次
"11" -> 3 次
"111" -> 1 次
```

**示例 2：**

```
输入：s = "101"
输出：2
解释：子字符串 "1" 在 s 中共出现 2 次
```

**示例 3：**

```
输入：s = "111111"
输出：21
解释：每个子字符串都仅由 '1' 组成
```

**示例 4：**

```
输入：s = "000"
输出：0
```

 

**提示：**

- `s[i] == '0'` 或 `s[i] == '1'`
- `1 <= s.length <= 10^5`

### 1、很简单的做法

C++,双百 为预防数据量过大，先进行结果的保存，结果发现数据量1不是很大...尴尬



~~~cpp

    int numSub(string s) {
        int len = s.size();
        int mod = 1e9 + 7,res = 0;
        vector<int> dp(len + 1, 0);
        for (int i = 1; i <= len; ++i) {
            dp[i] += (dp[i-1] +i)%mod ;
        }
        
        for(int low = 0, high =0; low<len;){          
            if(s[low] == '1'){
                high = low + 1;
                while( high<len && s[high] == '1') { ++high; } //统计连续1的个数              
                res += dp[high-low];
                res %= mod;
                low = high -1;
            }
            low++;
            
        }
        
        return res;       
    }
~~~

    int numSub(string s) {        
        int mod = 1e9+7,len = s.size();
 		int res = 0,count = 0;
        for(int i = 0; i < len; ++i){
            if(s[i] == '1'){
                ++count;
                res = ( res + count )%mod;
            }else{
                count = 0;
            }
        }       
        return res;    
    }
~~~
